HandlerMethodArgumentResolver

HandlerMethodArgumentResolver，是一个策略接口，目的是将给定Request中的方法参数解析为给定请求的上下文中的参数值。即可以自定义Request方法中的数据绑定，比如可以将用户信息置于session中，通过实现HandlerMethodArgumentResolver接口，实现session中数据和代码中的类进行数据绑定。

Strategy interface for resolving method parameters into argument values in the context of a given request.
